# SQL Data Cleaning Project

This project demonstrates data cleaning techniques using MySQL on the World Layoffs dataset from Kaggle. The cleaning process includes removing duplicate records, standardizing data formats, handling missing values, converting data types, and preparing the dataset for exploratory data analysis (EDA).

## Skills Used

* SQL (MySQL)
* Data Cleaning
* Window Functions
* Common Table Expressions (CTEs)
* Joins
* Data Standardization
* Date Conversion
* NULL Value Handling
